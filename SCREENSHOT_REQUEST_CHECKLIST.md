## üìã Screenshot Quality Guidelines

### Resolution & Size
- **Desktop:** Capture at 1440px or 1920px width
- **Tablet:** 768px or 1024px width
- **Mobile:** 375px or 414px width
- Use actual pixels (don't zoom in browser)

### Browser Setup
- Hide bookmarks bar
- Hide browser extensions
- Use incognito/private mode for clean capture
- Clear screenshot area (no browser UI in capture)

### What to Include
- Full section from top to bottom
- Surrounding context for spacing reference
- Multiple states (default, hover, active)
- Both light and dark variations (if exists)

### File Format
- PNG format (best quality)
- Descriptive naming (see naming convention above)
- Organize by section folder
