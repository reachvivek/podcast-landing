## Tips for Best Screenshots

1. **Full Resolution**: Take screenshots at actual size (don't zoom in/out)
2. **Clean Browser**: Hide bookmarks bar, extensions, etc.
3. **Multiple Angles**: Capture different states (default, hover, active)
4. **Context**: Include surrounding elements for spacing reference
5. **Mobile**: Use browser dev tools to capture responsive views
