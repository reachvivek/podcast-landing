# Hero Section Screenshots

## What to Capture

The hero section is the first thing visitors see - typically full-screen (100vh) with a large headline, CTA buttons, and background image/video.

### Required Screenshots:

1. **Full hero section** (desktop view)
   - Filename: `podover-hero-desktop-full.png`
   - What: Complete hero section from top of page to first scroll

2. **Hero text layout**
   - Filename: `podover-hero-text-layout.png`
   - What: Close-up of headline, subheadline, and CTA buttons
   - Note: Pay attention to text hierarchy, sizes, spacing

3. **CTA buttons** (primary and secondary)
   - Filename: `podover-hero-buttons.png`
   - What: Button styles, colors, sizes, spacing between buttons
   - Bonus: Capture hover state if possible

4. **Background treatment**
   - Filename: `podover-hero-background.png`
   - What: How image/video is treated (gradient overlay, blur, etc.)

5. **Mobile view**
   - Filename: `podover-hero-mobile.png`
   - What: How hero adapts to mobile (375px width)

### Design Elements to Note:

- [ ] Headline size and weight
- [ ] Subheadline style
- [ ] CTA button colors (we'll change to lime green)
- [ ] Button sizes and padding
- [ ] Text alignment (center/left)
- [ ] Background image opacity/overlay
- [ ] Any animations or effects
- [ ] Scroll indicator (arrow/mouse icon)
