### Design Elements to Note:

- [ ] Logo placement and size
- [ ] Navigation menu layout (centered, right-aligned)
- [ ] Menu item spacing
- [ ] Font size and weight for nav items
- [ ] Active/hover state for nav items
- [ ] CTA button style (color, size, placement)
- [ ] Background color (transparent vs solid)
- [ ] Sticky header height reduction
- [ ] Hamburger icon design
- [ ] Mobile menu animation (slide, fade, etc.)
- [ ] Shadow/border on sticky state
